# 🛡️ Fynz API - Backend Engine

Este repositorio contiene la API de alto rendimiento que impulsa la plataforma **Fynz**. Es el motor encargado de la lógica de negocio, seguridad y persistencia de datos financieros.

## 🌟 Visión General
El backend de Fynz está diseñado bajo principios de eficiencia y seguridad (Security by Design), ofreciendo:
- **Alta Disponibilidad:** Optimizado para despliegues en la nube.
- **Seguridad Extrema:** Autenticación JWT y validación estricta de cada entrada.
- **Escalabilidad:** Integración con PostgreSQL para un manejo robusto de datos.

## 🚀 Despliegue
Este proyecto se despliega automáticamente en **Render** mediante GitHub Actions al hacer push a la rama `main`.

## 🛠️ Stack Principal
- **Node.js** + **Fastify**
- **PostgreSQL** + **Kysely**
- **OpenAPI 3.0** (Documentación)
