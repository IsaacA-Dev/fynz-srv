# Fynz API

Backend para la gestión de finanzas personales.

## Despliegue
Este proyecto se despliega automáticamente en Render mediante GitHub Actions al hacer push a la rama `main`.
El motor de Fynz. Gestión eficiente de transacciones, ahorros y presupuestos personales. Stack ultra-ligero desplegado en DigitalOcean: Node + Fastify + SQLite.
