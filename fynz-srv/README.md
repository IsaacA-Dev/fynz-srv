# ⚙️ Fynz Server

Este repositorio contiene la API REST encargada de gestionar la lógica de negocio, autenticación y persistencia de datos de la plataforma Fynz. Es el núcleo que alimenta tanto el panel de administración como el dashboard financiero del cliente.

### 🔗 Frontend
El cliente web que consume este servicio se encuentra desplegado en: [https://isaaca-dev.github.io/Fynz/](https://isaaca-dev.github.io/Fynz/)

### 🛠️ Tecnologías clave
- **Servidor**: Node.js con Fastify para alta performance.
- **Autenticación**: JWT (JSON Web Tokens) para sesiones seguras.
- **Base de Datos**: Gestión eficiente de registros financieros y perfiles de usuario.
- **Documentación**: Especificación OpenAPI/Swagger para el diseño de endpoints.
